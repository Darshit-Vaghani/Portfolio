Project Overview

A full-featured real-time vehicle tracking system leveraging 4G GSM and GNSS. Data from the EC200UCN module is sent via Express.js/Node.js to a MongoDB server, tunneled over ngrok for anywhere access, and visualized on a Leaflet.js map.
