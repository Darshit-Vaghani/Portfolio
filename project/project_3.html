Project Overview

A safety-focused IoT system designed to detect LPG leaks using an MQ-6 sensor. Upon detecting a leak, the system automatically triggers a servo motor to shut off the gas valve, preventing further hazard. An audible buzzer and an automated night cutoff add extra layers of safety.
