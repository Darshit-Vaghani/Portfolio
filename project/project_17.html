Project Overview

A novel hands-free fan speed controller using clap gestures. The "Clap Of Thing" (COT) system detects 2, 3, or 4 claps to set predefined speeds or enable manual potentiometer control via TRIAC phase modulation.

Key Features

2 Claps → Speed Level 2

Medium fan speed activation

3 Claps → Speed Level 3

High fan speed activation

4 Claps → Manual Mode

Enables fine control via potentiometer

Achievements

Introduced COT — Intuitive Gesture-Based AC Control

Key Learnings

Clap Signal Processing

Implemented robust clap detection with noise filtering on limited-resource ATTiny85.

TRIAC Phase Control

Mastered zero-crossing and phase-angle modulation for smooth AC load control.

Compact PCB Design

Optimized circuit layout for minimal footprint and reliable high-voltage isolation.
